# mobile_app_ca
GDipSA 62 Mobile App Group 3 project

## 1. Project Overview
This project involves the development of an AI-Enabled Wellness Mobile Application.

## 2. Application Features
### 2.1 Main Features
A. Support user login and logout in the mobile application
B. Enter and view basic wellness records (example of wellness records - sleep hours,
exercise activities and duration etc.)
C. Connect to a backend service
D. Store and retrieve data from a MySQL database (through backend services)
E. Interact with a simple chatbot
